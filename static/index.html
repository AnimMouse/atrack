Bittrk Alpha 3

This is an experimental tracker.

You can use the following addresses to add the tracker to your torrents or ntrack apps:

  • http://bittrk.appspot.com/announce
  • https://bittrk.appspot.com/announce (For secure SSL connections)

Powered by the public domain Atrack tracker for Google App Engine.

For feedback email uriel@berlinblue.org or join #bittorrent in irc.freenode.org

Note: For privacy and efficiency reasons, atrack does not (yet?) implement the `scrape` protocol, so you wont see the number of seeders and leechers available in this tracker, this doesn't mean they aren't there, but atrack doesn't keep track (pun not intended ;)) of that information.
