Welcome to the (last and only?) running instance of Atrack,

a torrent tracker designed to run in Google App Engine.

Running instances:

Front-end Instance: https://1.track.ga/announce | Memcache

Back-end Instance: https://2.track.ga/announce | Memcache

All instances are running on Atrack v0.9.10 since June 8, 2018

Note: Back-end Instance only contains 9 hours of free usage, expect downtime.

Note: Clicking on the link above on a browser will just redirect you to this site, unless if the instance is experiencing over quata.

Note: This program is still in its beta stage and has not been updated since October 30, 2009.

Note: This program does not yet handle ipv6, because of that, AAAA records are removed.

Note: For privacy and efficiency reasons, atrack does not (yet?) implement the `scrape` protocol and other non-essential BT features, so you wont see the number of seeders and leechers available in this tracker, this doesn't mean they aren't there, but atrack doesn't keep track (pun not intended) of that information.
